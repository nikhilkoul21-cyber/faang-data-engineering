# FAANG Data Engineering Portfolio

This repository contains FAANG-level data engineering projects focused on
building scalable, cloud-native data platforms with strong emphasis on
data quality, governance, and cost efficiency.

- Advanced SQL and analytics

- Production-grade Python
- Data quality, governance, and lineage
- Cost-aware system design
